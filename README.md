# API Call 3 Ways

1. Restful API
   Pros

- CRUD Operation (GET POST PUT DELETE (FETCH))
- Complete separation between client and server
- Can recognize each API role by endpoint
  Cons
- Over fetching
- Under fetching

2. GraphQL
   Pros

- Solve over fetching
- Solve under fetching by one request
- Only one endpoint
  Cons
- File Upload
- Hard to filter wrong request

3. SOAP
